• How do I build the SpiderMonkey source for Mac OS X? I get an error 
    jsprf.c: In function 'BuildArgArray':
jsprf.c:644: error: incompatible types in assignment
    Add the following near the end of the file config.mk in the src/ directory 
    ifeq ($(OS_ARCH), Darwin)
 DEFINES += -DHAVE_VA_COPY -DVA_COPY=va_copy
endif
